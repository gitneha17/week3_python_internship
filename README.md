**📊 Dataset Description**

The dataset sales_data.csv contains information about product sales.

Column	Description
Product	Name of the product
Category	Product category
Sales	Sales amount

The dataset may contain:

Missing values
Duplicate records

These issues are handled during the data cleaning process.

**🧹 Data Cleaning Process**

The following steps were performed:

Load the dataset using Pandas

Remove missing values

Remove duplicate rows

Prepare the cleaned data for analysis

This ensures that the dataset is accurate and ready for analysis.

**🔢 NumPy Operations**

NumPy was used to perform numerical analysis on the sales data.

The following statistics were calculated:

Average sales

Maximum sales

Minimum sales

Total sales

This demonstrates the use of NumPy arrays for efficient numerical computation.

**📈 Data Analysis Using Pandas**

Pandas was used to analyze the dataset and generate insights.

Category-wise Analysis

The average sales were calculated for each product category.

Product-wise Analysis

Total sales for each product were calculated using groupby operations.
